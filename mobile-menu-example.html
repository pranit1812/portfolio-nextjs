Mobile Menu Example

This is a simple example of a working mobile menu. Resize the browser window to see how it adapts to different screen sizes.

On mobile devices, click the blue button in the top right corner to toggle the mobile menu.

Implementation Notes:

1. The mobile menu uses vanilla JavaScript for toggling visibility

2. CSS media queries handle responsive behavior

3. The menu has a gradient background with backdrop blur for a modern look

4. This example doesn't rely on any framework, making it easy to adapt to Next.js
